# Hospital Data Analysis Project

## Overview
This project performs exploratory data analysis (EDA) on hospital patient data to extract meaningful insights.

## Features
- Data cleaning and preprocessing
- Patient demographics analysis
- Billing trends visualization
- Statistical analysis (T-test)
- Correlation analysis

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas, NumPy, Seaborn, Matplotlib, Scipy

## 📈 Key Insights
- Identified the top 5 hospitals by total revenue.
- Visualized that certain medical conditions have higher frequency in specific age groups.
- Statistical testing confirmed whether gender plays a significant role in billing amounts.
